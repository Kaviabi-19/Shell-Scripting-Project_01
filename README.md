# 📄 AWS Resource Listing Script

This Bash script automates the process of listing resources from various AWS services using the AWS CLI. It allows users to quickly fetch resource details by specifying the AWS region and service name as inputs.

# 🚀 Features
Supports multiple AWS services like EC2, RDS, S3, Lambda, DynamoDB, and more
Simple command-line usage
Validates AWS CLI installation and configuration before execution
# ⚙️ Prerequisites
AWS CLI installed
AWS CLI configured (aws configure)
# ▶️ Usage
    ./aws_resource_list.sh <aws_region> <aws_service>
# 📌 Example
    ./aws_resource_list.sh us-east-1 ec2
# 🧩 Supported Services

EC2, RDS, S3, CloudFront, VPC, IAM, Route53, CloudWatch, CloudFormation, Lambda, SNS, SQS, DynamoDB, and EBS.
